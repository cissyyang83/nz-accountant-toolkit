# Changelog

All notable changes to NZ Accountant Toolkit will be recorded here.

## v0.1.0 - 2026-06-07

### Added

- Initial open-source repository structure.
- Main README with overview, features, usage examples, roadmap, contribution guide, and disclaimer.
- Python helper package with FIF, GST variance, and quick sale calculation functions.
- Command-line interface for selected calculator helpers.
- Automated tests and GitHub Actions workflow.
- Governance, maintainer, security, support, privacy, maintenance, and release checklist files.
- Template specifications for FIF, quick sale, GST reconciliation, property development costs, shareholder current accounts, and year-end workpapers.
- Checklists for GST review and year-end accounts.
- Xero bank reconciliation SOP.
- Examples using New Zealand GST, FIF, FDR, CV, quick sale adjustment, property development, shareholder loans, and trust accounting concepts.
- Starter GitHub issue list.
- GitHub discussion topic suggestions.
- Release v0.1 notes.
