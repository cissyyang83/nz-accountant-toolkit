# Privacy

NZ Accountant Toolkit is designed to avoid collecting or storing client information.

## Repository Data

This repository should contain only:

- Documentation.
- Template specifications.
- Example data.
- Test data.
- Source code.

It should not contain:

- Client names.
- IRD numbers.
- Bank account numbers.
- Xero organisation identifiers.
- Email addresses from client files.
- Real financial statements.
- Real tax returns.

## Example Data

All examples should be fictional, rounded, and clearly marked as sample data.

## Future Xero Integration

Any future Xero integration should document:

- What data is requested.
- Why it is needed.
- Where it is stored.
- How credentials are protected.
- How users can disconnect access.

## Contributor Responsibility

Contributors must remove confidential information before opening issues, discussions, or pull requests.
