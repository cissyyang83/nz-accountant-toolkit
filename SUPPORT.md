# Support

NZ Accountant Toolkit is a volunteer open-source project.

## Getting Help

Use GitHub Issues for:

- Bugs in calculator logic.
- Problems with examples.
- Missing review prompts.
- Documentation corrections.
- Suggested templates or checklists.

Use GitHub Discussions for:

- General questions.
- Template ideas.
- Workflow suggestions.
- Practitioner feedback.

## Not Supported

The project does not provide:

- Personal tax advice.
- Legal advice.
- Financial advice.
- Inland Revenue filing support.
- Client-specific conclusions.

## Before Asking

Please include:

- The file or tool involved.
- The accounting context.
- Expected result.
- Actual result.
- Confirmation that no confidential client data is included.
