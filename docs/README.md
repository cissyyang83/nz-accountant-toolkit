# Documentation

This folder contains practitioner-focused documentation for NZ Accountant Toolkit.

The documents are written for accountants, tax agents, bookkeepers, and small business advisors. They explain when to use each tool, what information is needed, what assumptions should be reviewed, and what evidence should be retained.
