# GitHub Discussions Topics

Suggested GitHub Discussions categories and starter topics.

## Announcements

Use for release notes, roadmap updates, and maintainer announcements.

Starter topics:

- Welcome to NZ Accountant Toolkit.
- Release v0.1: documentation-first launch.
- Roadmap feedback for Phase 1 Excel templates.

## General

Use for broad questions about the project and suggested improvements.

Starter topics:

- Which New Zealand accounting templates would help your practice most?
- What should be included in a small-firm year-end workpaper pack?
- How should we keep the toolkit useful for both accountants and bookkeepers?

## Template Ideas

Use for new Excel workbook and checklist ideas.

Starter topics:

- Suggested fields for the FIF Calculator workbook.
- GST reconciliation layout feedback.
- Property development cost allocation approaches.
- Shareholder current account tracker improvements.

## Tax and Accounting Review

Use for professional review of wording, assumptions, and risk prompts.

Starter topics:

- FIF FDR and CV explanation review.
- Quick sale and land tax review prompts.
- Trust accounting checklist scope.
- GST risk areas commonly missed in practice.

## Development

Use for future Python calculators, web app design, and Xero integration.

Starter topics:

- Preferred CSV format for calculator inputs.
- Plain-English output for Python calculators.
- Web app workflow for accountant review.
- Xero integration privacy and permission concerns.

## Good First Issues

Use to help new contributors find approachable tasks.

Starter topics:

- Add one realistic GST example.
- Improve one checklist section.
- Add reviewer prompts to one template.
- Review spelling and New Zealand terminology.
