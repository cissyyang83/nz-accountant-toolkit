# Project Scope

NZ Accountant Toolkit is an open-source collection of practical resources for New Zealand accounting and tax work.

## Intended Users

- Accountants preparing annual accounts and income tax returns.
- Tax agents reviewing GST, FIF, property, and shareholder loan positions.
- Bookkeepers reconciling Xero files.
- Small business advisors supporting owner-managed businesses.
- Trustees and advisors reviewing trust accounting records.

## Current Scope

The first release focuses on:

- Spreadsheet design documents.
- Review checklists.
- Standard operating procedures.
- Example scenarios.
- Roadmap and open-source project structure.

## Out of Scope

The current release does not provide:

- Finished Excel workbooks.
- Tax advice.
- Legal advice.
- Financial advice.
- Automated Inland Revenue filing.
- Automated Xero data extraction.

## Review Philosophy

Each tool should help a practitioner:

1. Identify the client fact pattern.
2. Gather supporting evidence.
3. Apply a consistent calculation or review method.
4. Record assumptions.
5. Highlight issues for senior review.
6. Retain a clean workpaper trail.
