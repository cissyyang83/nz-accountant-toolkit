# Release v0.1 Notes

## Release Summary

NZ Accountant Toolkit v0.1 is the first documentation-led release of the project.

This release establishes the repository as a practical open-source project for New Zealand accountants, tax agents, bookkeepers, and small business advisors.

## Included

- Professional repository structure.
- Main README and governance files.
- Documentation folder.
- Template specification folder.
- Tool documentation folder.
- Examples folder.
- Checklists folder.
- Xero workflow folder.
- Property development folder.
- Roadmap for Excel templates, Python calculators, web application, and Xero integration.
- Starter GitHub issues.
- GitHub Discussions topic suggestions.

## Key Practitioner Topics

- GST reconciliation.
- FIF review using FDR and CV concepts.
- Quick sale property adjustment.
- Property development costs.
- Shareholder current accounts.
- Xero bank reconciliation.
- Year-end workpapers.
- Trust accounting.

## Known Limitations

- No finished Excel files are included yet.
- No Python calculators are included yet.
- No web application is included yet.
- No Xero API integration is included yet.
- All tax positions require practitioner review.

## Next Release Target

The next release should focus on Phase 1 Excel templates, beginning with:

- FIF Calculator workbook.
- GST Reconciliation workbook.
- Shareholder Current Account Tracker workbook.
