# Roadmap

NZ Accountant Toolkit will be developed in four practical phases.

## Phase 1: Excel Templates

Goal: Convert template specifications into usable Excel workbooks for accountants.

Planned work:

- FIF Calculator workbook.
- Quick Sale Calculator workbook.
- GST Reconciliation workbook.
- Property Development Cost Tracker workbook.
- Shareholder Current Account Tracker workbook.
- Year-End Workpaper Checklist workbook.
- Sample data tabs for training and review.
- Protected formula cells and input guidance.

## Phase 2: Python Calculators

Goal: Build tested calculators that can process simple CSV or manual inputs.

Planned work:

- FIF FDR and CV calculator.
- Quick sale adjustment calculator.
- GST variance checker.
- Shareholder current account balance checker.
- Property development margin calculator.
- Automated test cases using New Zealand examples.
- Plain-English command output.

## Phase 3: Web Application

Goal: Provide a simple browser-based interface for accountants and bookkeepers.

Planned work:

- Guided FIF review workflow.
- GST reconciliation dashboard.
- Property development project tracker.
- Shareholder current account dashboard.
- PDF summary reports.
- Reviewer sign-off screens.
- User-friendly export formats.

## Phase 4: Xero Integration

Goal: Explore secure integration with Xero for reconciliation and review workflows.

Planned work:

- Xero authentication research.
- Bank reconciliation exception reports.
- GST control account extraction.
- Trial balance import.
- Transaction coding review.
- Privacy and security guidance.
- Firm-level workflow documentation.

## Long-Term Ideas

- Inland Revenue checklist mapping.
- Template version comparison.
- Practice management integrations.
- Training examples for junior accountants.
- Multi-entity group workpapers.
- Trust distribution workflow.
