# Template Design Standards

These standards apply to future Excel templates and current template specifications in this repository.

## Workbook Structure

Each workbook should include:

- Cover sheet.
- Instructions sheet.
- Input sheet.
- Calculation sheet.
- Review sheet.
- Output summary.
- Version history.

## Colour Convention

Suggested colour convention:

| Colour | Meaning |
| --- | --- |
| Light yellow | User input |
| Light blue | Linked data |
| Light green | Output |
| Light grey | Formula or locked cell |
| Red text | Exception or review warning |

## Review Requirements

Each template should include:

- Client name.
- Balance date.
- Tax year.
- Prepared by.
- Reviewed by.
- Date prepared.
- Date reviewed.
- Assumptions section.
- Evidence section.
- Sign-off section.

## Formula Design

Formulas should be:

- Visible enough for review.
- Avoided in input cells.
- Protected where appropriate.
- Supported by cross-checks.
- Documented where the logic is not obvious.

## Version Control

Each template should include:

- Template name.
- Version number.
- Release date.
- Change summary.
- Known limitations.
