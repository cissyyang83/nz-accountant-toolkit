# FIF Example

## Scenario

A New Zealand tax resident individual holds a United States ETF through an online broker.

| Item | Amount |
| --- | ---: |
| Opening market value | NZD 80,000 |
| Closing market value | NZD 92,000 |
| Purchases during year | NZD 10,000 |
| Sales proceeds | NZD 0 |
| Dividends received | NZD 1,600 |

## FDR Design Calculation

```text
80,000 x 5% = 4,000
```

## CV Design Calculation

```text
92,000 + 0 + 1,600 - 80,000 - 10,000 = 3,600
```
