# GST Reconciliation Example

## Scenario

A GST-registered New Zealand consulting business files GST every two months on the payments basis.

| Source | GST Payable |
| --- | ---: |
| Xero GST report | NZD 12,450 |
| Filed Inland Revenue return | NZD 12,300 |
| Difference | NZD 150 |

## Possible Explanation

The difference may relate to a late-coded supplier invoice or a manual GST adjustment made before filing.

## Workpaper Steps

1. Compare Xero GST audit report to filed return.
2. Check manual adjustments.
3. Review transactions coded after the return was filed.
4. Reconcile GST control account.
5. Record explanation and reviewer sign-off.
