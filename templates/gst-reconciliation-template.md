# GST Reconciliation Template Specification

## Purpose

Design an Excel workbook to reconcile GST return amounts to accounting records and Inland Revenue filing history.

## Intended Use

Use this template for GST-registered clients using Xero or similar accounting software.

## Workbook Sheets

| Sheet | Purpose |
| --- | --- |
| Cover | Client, GST basis, filing frequency, preparer, reviewer |
| GST Returns | Filed return periods and amounts |
| Xero GST Reports | Output and input tax by period |
| GST Control Accounts | General ledger movement |
| IRD Statement | Filing and payment history |
| Adjustments | Private use, mixed-use, entertainment, timing |
| Reconciliation | Variance analysis |
| Review Summary | Exceptions and sign-off |

## Key Fields

- GST registration status.
- GST basis: invoice, payments, or hybrid.
- Filing frequency.
- Period start and end dates.
- Sales GST.
- Purchases GST.
- Adjustments.
- GST payable or refundable.
- GST control account balance.
- Inland Revenue return amount.
- Payment or refund date.

## Reconciliation Checks

- GST returns agree to accounting records.
- GST control account clears after payment or refund.
- Timing differences are explained.
- Balance sheet GST balance is reasonable.
- GST on asset purchases and disposals is reviewed.
- Private use and non-deductible items are considered.
- Zero-rated and exempt supplies are supported.

## Example Scenario

| Source | GST Payable |
| --- | ---: |
| Xero GST report | NZD 12,450 |
| Filed GST return | NZD 12,300 |
| Difference | NZD 150 |

Review note:

The NZD 150 difference may relate to a manual adjustment, late-coded invoice, or timing difference. Preparer should identify the transaction and retain support.
