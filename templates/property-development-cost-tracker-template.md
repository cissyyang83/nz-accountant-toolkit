# Property Development Cost Tracker Template Specification

## Purpose

Design an Excel workbook to track costs, GST treatment, funding, and sale proceeds for New Zealand property development projects.

## Intended Use

Use this template for builders, developers, landowners undertaking subdivision, and clients with mixed capital and revenue property activity.

## Workbook Sheets

| Sheet | Purpose |
| --- | --- |
| Cover | Project, entity, tax year, preparer, reviewer |
| Project Summary | Site, stages, expected completion, GST status |
| Land Costs | Purchase price, legal fees, due diligence |
| Development Costs | Consultants, consents, engineering, construction |
| Holding Costs | Rates, insurance, interest, security |
| Funding | Loans, shareholder advances, repayments |
| GST Tracking | GST claimable, GST payable, zero-rated items |
| Sales | Lot sales, deposits, settlement adjustments |
| Cost Allocation | Allocation by lot or stage |
| Profit Summary | Project margin and tax workpaper summary |
| Review Notes | Risks, assumptions, sign-off |

## Cost Categories

- Land acquisition.
- Legal and conveyancing.
- Resource consent.
- Surveying.
- Engineering.
- Earthworks.
- Construction.
- Professional fees.
- Finance costs.
- Rates and insurance.
- Marketing and agent fees.
- Holding costs.

## Review Prompts

- Is the activity a taxable property development?
- Has GST registration been considered?
- Were land transactions zero-rated where required?
- Are costs allocated consistently across lots?
- Are shareholder advances reconciled?
- Are related-party costs documented?
- Is work in progress complete and supported?

## Example Scenario

| Category | Amount |
| --- | ---: |
| Land | NZD 1,200,000 |
| Consents and professional fees | NZD 180,000 |
| Construction | NZD 2,400,000 |
| Holding and finance costs | NZD 210,000 |
| Sale proceeds | NZD 4,600,000 |

Draft project margin:

```text
4,600,000 - 1,200,000 - 180,000 - 2,400,000 - 210,000 = 610,000
```

Reviewer should confirm GST and income tax treatment before finalising.
