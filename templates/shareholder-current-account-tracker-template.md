# Shareholder Current Account Tracker Template Specification

## Purpose

Design an Excel workbook to track shareholder current account movements for New Zealand companies.

## Intended Use

Use this template where shareholders or associated persons receive advances, make repayments, receive dividends or salaries, or have overdrawn current account balances.

## Workbook Sheets

| Sheet | Purpose |
| --- | --- |
| Cover | Company, shareholder, tax year, preparer, reviewer |
| Opening Balances | Prior year closing balances |
| Transactions | Advances, repayments, expenses, dividends, salaries |
| Dividend Summary | Dividends, resident withholding tax, imputation credits |
| Salary Summary | Shareholder salaries and PAYE notes |
| Interest Review | Interest charged or fringe benefit review |
| Closing Position | Balance by shareholder |
| Review Notes | Risks, resolutions, sign-off |

## Key Fields

- Shareholder name.
- Opening current account balance.
- Drawings.
- Personal expenses paid by company.
- Repayments.
- Dividends credited.
- Shareholder salary credited.
- PAYE or withholding tax.
- Imputation credits.
- Interest charged.
- Closing balance.

## Review Checks

- Are overdrawn balances identified?
- Are shareholder salaries approved and supported?
- Are dividends supported by solvency and director resolutions?
- Are imputation credits recorded correctly?
- Has interest or fringe benefit tax been considered?
- Are personal expenses coded consistently?
- Are associated person balances included where needed?

## Example Scenario

| Item | Amount |
| --- | ---: |
| Opening balance owing to shareholder | NZD 12,000 |
| Drawings | NZD 48,000 |
| Repayments | NZD 8,000 |
| Dividend credited | NZD 30,000 |
| Closing balance | NZD -10,000 |

The negative balance indicates the shareholder may owe the company NZD 10,000. Reviewer should consider documentation, repayments, interest, and tax consequences.
