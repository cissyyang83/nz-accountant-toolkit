# Year-End Workpaper Checklist Template Specification

## Purpose

Design an Excel workbook to guide annual accounts and income tax return preparation for New Zealand clients.

## Intended Use

Use this template for companies, trusts, sole traders, partnerships, and rental property clients.

## Workbook Sheets

| Sheet | Purpose |
| --- | --- |
| Cover | Client, entity type, balance date, preparer, reviewer |
| Information Request | Client records requested and received |
| Trial Balance Review | Key account checks |
| Tax Adjustments | Add-backs, deductions, timing adjustments |
| GST Review | GST reconciliation and exceptions |
| Fixed Assets | Additions, disposals, depreciation |
| Loans and Interest | Bank loans, shareholder loans, related parties |
| Trust Accounting | Beneficiary allocations and resolutions |
| Final Review | Senior review and filing sign-off |

## Key Review Areas

- Bank reconciliation.
- Accounts receivable and payable.
- GST control accounts.
- PAYE and payroll.
- Fixed assets.
- Loans and interest.
- Shareholder current accounts.
- Trust beneficiary accounts.
- Entertainment and private use.
- Home office and motor vehicle claims.
- Tax losses and imputation credits.

## Output

The template should produce:

- Completion status summary.
- Outstanding information list.
- Tax adjustment summary.
- Reviewer sign-off.
- Filing readiness status.
