# Tools

This folder documents practical calculator and workflow tools for NZ Accountant Toolkit.

The current release provides design documentation and a small Python helper layer for selected calculations. Future phases will expand the calculators and add a web application.

## Command Examples

```bash
nz-accountant-toolkit fif --opening 80000 --closing 92000 --purchases 10000 --dividends 1600
nz-accountant-toolkit gst-variance --accounting 12450 --filed 12300
nz-accountant-toolkit quick-sale --sale-price 900000 --purchase-price 760000 --acquisition-costs 7500 --improvements 48000 --sale-costs 22000
```
