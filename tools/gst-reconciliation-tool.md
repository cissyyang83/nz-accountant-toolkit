# GST Reconciliation Tool

## Purpose

The GST Reconciliation Tool supports reconciliation between GST returns, accounting records, and Inland Revenue records.

## Inputs

- GST return periods.
- GST basis.
- Sales GST.
- Purchases GST.
- Adjustments.
- Filed GST return amounts.
- GST control account movement.
- Inland Revenue payment or refund details.

## Outputs

- Period-by-period variance report.
- GST control account reconciliation.
- Outstanding differences.
- Review notes and sign-off.

## Common Exceptions

- Late-coded invoices.
- Manual GST return adjustments.
- Private use adjustments.
- Entertainment add-backs.
- Asset purchases or disposals.
- GST on imported services.
- Timing differences between Xero and Inland Revenue.
